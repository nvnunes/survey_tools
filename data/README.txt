==========================
    NOTE ON DATA FILES
==========================

Certain data files required by specific tools (e.g., "sky.py") are not 
included in this package due to licensing restrictions that prevent 
distribution under the MIT license.

If you need access to these files, please contact the package author 
and include the following information in your request:

- Your name
- Your position
- Your institution
- A brief description of how you intend to use the data
